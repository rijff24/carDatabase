# Car Sales and Repair Tracking System

A web-based application for tracking car sales, inventory, and repairs.

## Features

- Dashboard with key metrics
- Car inventory management
- Repair tracking
- Sales recording and monitoring
- Reporting and analytics

## Technical Stack

- Python (Backend)
- Flask (Web Framework)
- SQLite/PostgreSQL (Database)
- HTML/CSS/JavaScript (Frontend)
- Bootstrap (UI Framework)

## Dashboard

The dashboard provides an overview of:
- Total cars in inventory
- Cars currently for sale
- Recent sales statistics
- Average profit
- Cars waiting for repair
- Cars on display stands the longest
- Recent completed repairs

## Installation

1. Clone the repository
```
git clone https://github.com/rijff24/carDatabase.git
```

2. Install dependencies
```
pip install -r requirements.txt
```

3. Set up the database
```
python init_db.py
```

4. Run the application
```
python app.py
```
